# 맞춤법 검사기 Frontend

웹 버전(데모용)과 데스크톱용이 있습니다.
웹 버전은 서버에 저장된 설정만으로 검사할 수 있으며, 데스크톱용은 자유롭게 단어를 추가하고, 파일 단위로 검사할 수 있습니다.

## run 방법

npm run dev:desktop

npm run dev:web

## 빌드 방법

npm run build

버전을 지정하지 않았을 시 데스크톱 버전과 웹 버전 둘 다 빌드합니다.

npm run build:desktop

데스크톱 버전을 빌드합니다.

npm run build:web

웹 버전을 빌드합니다.
